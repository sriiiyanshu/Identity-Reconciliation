# Identity Reconciliation Tool

This is an assignment given by BiteSpeed as a part of their assessment for the Engineering Intern position.

The assignement was made with the help of Github Copilot.

To run this project locally:

1. Install the libraries

```
npm i
```

2. Run the backend and frontend together
```
npm run dev
```

### Backend is configured to run at port 3000 
### Frontend is at 5173 (standard react application)

The backend code is in the server directory whereas the basic frontend is at the client.

For any direct testing we can use postman at port 3000/identify


### Database used is PostgreS through NEONDB and Prisma ORM
